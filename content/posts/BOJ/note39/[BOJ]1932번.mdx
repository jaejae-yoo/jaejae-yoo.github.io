> ## [BOJ] 1932번 

#### 문제
    예시
            7
          3   8
        8   1   0
      2   7   4   4
    4   5   2   6   5

위 그림은 크기가 5인 정수 삼각형의 한 모습이다.

맨 위층 7부터 시작해서 아래에 있는 수 중 하나를 선택하여 아래층으로 내려올 때, 이제까지 선택된 수의 합이 최대가 되는 경로를 구하는 프로그램을 작성하라. 아래층에 있는 수는 현재 층에서 선택된 수의 대각선 왼쪽 또는 대각선 오른쪽에 있는 것 중에서만 선택할 수 있다.

삼각형의 크기는 1 이상 500 이하이다. 삼각형을 이루고 있는 각 수는 모두 정수이며, 범위는 0 이상 9999 이하이다.

##### 입력
첫째 줄에 삼각형의 크기 n(1 ≤ n ≤ 500)이 주어지고, 둘째 줄부터 n+1번째 줄까지 정수 삼각형이 주어진다.
##### 출력
첫째 줄에 합이 최대가 되는 경로에 있는 수의 합을 출력한다.

#### **풀이**
처음 찾아냈던 규칙은 answer[i][0] 과 answer[i][-1]을 제외한 나머지 인덱스라면, 이전에 저장했던 answer[i-1]의 해당 인덱스 기준 왼쪽에 있는 값들을 하나씩 이동하면서 필요한 수만큼 더해서 모든 값들을 저장하는 식으로 생각했다. 
아이디어는 맞는 것 같은데, 채점하면서 중간에 시간 초과가 뜨거나 에러가 나서 다시 생각해야 했다.

결국 구글에 검색하여 아이디어를 참고하니, answer[i][0] 과 answer[i][-1]을 제외한 나머지 인덱스라면 하나의 값에서 받을 수 있는 경우의 수는 2가지(왼쪽에서 오는 경우, 오른쪽에서 오는 경우)인 것을 활용하여 풀 수 있는 문제였다.

** [코드] **

    n = int(input())
    answer = []
    for i in range(n):
        answer.append(list(map(int, input().split())))

    for i in range(1, n):
        for j in range(i+1):
        if j == 0:
            answer[i][j] = answer[i][j] + answer[i-1][j]
        elif i == j:
            answer[i][j] = answer[i][j] + answer[i-1][j-1]
        else:
            answer[i][j] = max(answer[i-1][j-1], answer[i-1][j]) + answer[i][j]
    print(max(answer[n - 1]))
