> ## [BOJ] 9095번 
 
#### 문제
정수 4를 1, 2, 3의 합으로 나타내는 방법은 총 7가지가 있다. 합을 나타낼 때는 수를 1개 이상 사용해야 한다.

    1+1+1+1
    1+1+2
    1+2+1
    2+1+1
    2+2
    1+3
    3+1

정수 n이 주어졌을 때, n을 1, 2, 3의 합으로 나타내는 방법의 수를 구하는 프로그램을 작성하시오.
##### 입력
첫째 줄에 테스트 케이스의 개수 T가 주어진다. 각 테스트 케이스는 한 줄로 이루어져 있고, 정수 n이 주어진다. n은 양수이며 11보다 작다.
##### 출력
각 테스트 케이스마다, n을 1, 2, 3의 합으로 나타내는 방법의 수를 출력한다.
#### **풀이**
이 문제는 도저히 풀이가 떠오르지 않아, 구글에 검색하여 아이디어를 얻었다.
앞으로 제공된 수을 합하여 target 숫자를 만들 수 있는 방법의 수를 구하는 문제는, 규칙성을 찾는 것을 첫 번째 접근 방법으로 행해야겠다.

이 문제의 예시로 들면 1,2,3 으로 타켓 숫자를 만들 수 있는 방법은

    1 = 1 (1가지)
    2 = (1+1), 2 (2가지)
    3= (1+1+1), (2+1), (1+2) ,3 (4가지)
    4 = (1+1+1+1), (2+2), (1+3), (3+1), (1+1+2), (1+2+1),(2+1+1) (7가지)
    5 =  • • • • (13가지)

⇒ 타겟 숫자가 3보다 크면 (n-1)의 방법의 수+(n-2)의 방법의 수+(n-3)의 방법의 수라는 규칙성을 찾을 수 있다.


** [코드] **

    n = int(input())
    _list=[]
    for i in range(n):
      _list.append(int(input()))

    for num in _list:
      sum_list=[0,1,2,4]
      if num > 3:
        for j in range(4,num+1):
          sum_list.append(sum_list[j-3]+sum_list[j-2]+sum_list[j-1])
        print(sum_list[-1])
      else:
        print(sum_list[num])
