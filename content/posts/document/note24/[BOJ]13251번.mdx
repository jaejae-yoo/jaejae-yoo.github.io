> ## [BOJ] 13251번 

#### 문제
효빈이의 비밀 박스에는 조약돌이 N개 들어있다. 조약돌의 색상은 1부터 M까지 중의 하나이다.

비밀 박스에서 조약돌을 랜덤하게 K개 뽑았을 때, 뽑은 조약돌이 모두 같은 색일 확률을 구하는 프로그램을 작성하시오. 

##### 입력
첫째 줄에 M (1 ≤ M ≤ 50)이 주어진다.

둘째 줄에는 각 색상의 조약돌이 몇 개 있는지 주어진다. 각 색상의 조약돌 개수는 1보다 크거나 같고 50보다 작거나 같은 자연수이다.

셋째 줄에는 K가 주어진다. (1 ≤ K ≤ N)
##### 출력
첫째 줄에 뽑은 조약돌이 모두 같은 색일 확률을 출력한다. 정답과의 절대/상대 오차는 10-9까지 허용한다.
#### **풀이**

조건부확률 개념 생각하며 풀었다.
    ex) P(초록색|K)
    전체 조약돌에서 K개를 뽑았는데, 초록색일 확률
    P(초록색 ∩ 전체에서 K개 뽑음)/P(전체에서 K개 뽑음) 

예전에 itertools 라이브러리 사용해서 풀었는데 메모리초과떠서 나중에 풀어야지 하고 오늘 풀었다.


** [코드] **

    M = int(input())
    ans = list(map(int, input().split()))
    K = int(input())

    def comb(n, m):
      num=0
      num = n-m
      _cnt = 1
      for i in range(n,1,-1):
        if i == num:
          break
        _cnt *=i
      return _cnt

    deno = comb(sum(ans),K)
    cnt =[]
    for num in ans:
      if num >= K:
        s=0
        s = comb(num,K)
        cnt.append(s)
    print(sum(cnt)/deno)
