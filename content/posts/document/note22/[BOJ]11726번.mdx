> ## [BOJ] 11726번 

#### 문제
2×n 크기의 직사각형을 1×2, 2×1 타일로 채우는 방법의 수를 구하는 프로그램을 작성하시오.

아래 그림은 2×5 크기의 직사각형을 채운 한 가지 방법의 예이다.

##### 입력
첫째 줄에 n이 주어진다. (1 ≤ n ≤ 1,000)
##### 출력
첫째 줄에 2×n 크기의 직사각형을 채우는 방법의 수를 10,007로 나눈 나머지를 출력한다.
#### **풀이**

이 문제에서는 타일의 구성이 같다고 해도 타일의 위치가 다르면 다른 방법이다.
2x1 ~ 2x5인 경우까지 구해본 결과 규칙을 발견할 수 있어서 f[n]=f[n-1]+f[n-2] 점화식을 통해 해결할 수 있었다.

10,007로 나누어 주는 이유는 오버플로우를 방지하기 위함이다.

** [코드] **

    n = int(input())
    answer=[1,2]
    if n == 1:
      print(1)
    elif n == 2:
      print(2)
    else:
      for num in range(2,n):
        answer.append(answer[num-1]+answer[num-2])
      print(answer[-1]%10007)
