> ## [BOJ] 15649번 

#### 문제
자연수 N과 M이 주어졌을 때, 아래 조건을 만족하는 길이가 M인 수열을 모두 구하는 프로그램을 작성하시오.

1부터 N까지 자연수 중에서 중복 없이 M개를 고른 수열
##### 입력
첫째 줄에 자연수 N과 M이 주어진다. (1 ≤ M ≤ N ≤ 8)
##### 출력
한 줄에 하나씩 문제의 조건을 만족하는 수열을 출력한다. 중복되는 수열을 여러 번 출력하면 안되며, 각 수열은 공백으로 구분해서 출력해야 한다.

수열은 사전 순으로 증가하는 순서로 출력해야 한다.
#### **풀이**

중복 순열은 itertools 라이브러리의 product 함수를 사용해서 해결할 수 있었다.

핵심은 순열에 같은 숫자가 포함되지 않게 하는 것이다. 이것은 set 함수를 이용해서 중복되는 숫자가 있는지 확인할 수 있었다.

** [코드] **

    from itertools import product
    N, M= map(int, input().split(" "))

    if M == 1:
      for number in product([num for num in range(1, N+1)], repeat=M):
        print(number[0])
    else:
      for number in product([num for num in range(1, N+1)], repeat=M):
          s=set()
          for i in number:
            s.add(i)
          if len(s) == M:
            print(''.join(str(number)[1:-1].split(",")))
          else:
            pass
