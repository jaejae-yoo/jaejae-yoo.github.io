> ## [BOJ] 1912번 

#### 문제
n개의 정수로 이루어진 임의의 수열이 주어진다. 우리는 이 중 연속된 몇 개의 수를 선택해서 구할 수 있는 합 중 가장 큰 합을 구하려고 한다. 단, 수는 한 개 이상 선택해야 한다.

예를 들어서 10, -4, 3, 1, 5, 6, -35, 12, 21, -1 이라는 수열이 주어졌다고 하자. 여기서 정답은 12+21인 33이 정답이 된다.


##### 입력
첫째 줄에 정수 n(1 ≤ n ≤ 100,000)이 주어지고 둘째 줄에는 n개의 정수로 이루어진 수열이 주어진다. 수는 -1,000보다 크거나 같고, 1,000보다 작거나 같은 정수이다.
##### 출력
첫째 줄에 답을 출력한다.
#### **풀이**

수열에서 순서대로 숫자 하나씩 확인하며 cnt(수들의 합을 저장하는 변수)의 값을 업데이트
    1. 해당 수가 양수이다. ➜ cnt 변수에 더함.
    2. 해당 수가 음수인 경우
       우선 지금까지의 합(cnt)를 list에 저장해둔다.(초기화할 수도 있으므로.)
       2-1. 만약 지금까지의 합에 현재 확인하는 수를 더하였을 때 0보다 크지 않는다면, 이전의 합(cnt)들을 초기화함(0)
       2-2. 만약 지금까지의 합에 현재 확인하는 수를 더하였을 때 0보다 크다면, 현재 수를 더함
다이나믹 프로그래밍 방식으로 간결하게 풀 수 있는 문제인데, 그냥 구현해버렸다.

아직 다이나믹 프로그래밍 방식이 손에 익지 않는다.

** [코드] **

    n = map(int, input().split(" "))
    num = list(map(int, input().split(" ")))
    ans=[]
    if num[0] > 0:
      ans=[num[0]]
    if len(ans)>0:
      cnt=ans[0]
    else:
      cnt=0

    for i in range(1,len(num)):
      if num[i] > 0:
        cnt+=num[i]
      else:
        if cnt !=0:
          ans.append(cnt)
        if cnt + num[i] > 0:
          cnt +=num[i]
        else:
          cnt=0
    if cnt !=0:
      ans.append(cnt)
    ans.append(max(num))
    print(max(ans))
