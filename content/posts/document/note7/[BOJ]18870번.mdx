> ## [BOJ] 18870번 

#### 문제
수직선 위에 N개의 좌표 X1, X2, ..., XN이 있다. 이 좌표에 좌표 압축을 적용하려고 한다.

Xi를 좌표 압축한 결과 X'i의 값은 Xi > Xj를 만족하는 서로 다른 좌표의 개수와 같아야 한다.

X1, X2, ..., XN에 좌표 압축을 적용한 결과 X'1, X'2, ..., X'N를 출력해보자.
##### 입력
첫째 줄에 N이 주어진다.

둘째 줄에는 공백 한 칸으로 구분된 X1, X2, ..., XN이 주어진다.
##### 출력
첫째 줄에 X'1, X'2, ..., X'N을 공백 한 칸으로 구분해서 출력한다.
##### 제한
1 ≤ N ≤ 1,000,000

-109 ≤ Xi ≤ 109

#### **풀이**
둘째 줄에서 입력받은 좌표를 정렬하고 cnt=0부터 시작해서, 딕셔너리에 좌표가 존재하면 pass하고 존재하지 않는다면 새로운 좌표를 key값으로 지정하여 cnt를 value 값으로 추가한다.
그 후 위 딕셔너리에 저장된 값들을 기반으로 압축된 좌표를 출력한다.

문제만 읽었을 때는 복잡할 줄 알았는데 한번에 풀려서 기분좋았다 !

** [코드] **

    n = int(input())
    point = list(map(int, input().split(" ")))
    point_sort = sorted(point)
    cnt = 0
    _hash={}
    answer = []

    for num1 in point_sort:
      if num1 in _hash:
        pass
      else:
        _hash[num1] = cnt
        cnt+=1

    for num2 in point:
      print(_hash[num2], end=" ")
