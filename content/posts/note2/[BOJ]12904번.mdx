> ## [BOJ] 12904번 

#### 문제
수빈이는 A와 B로만 이루어진 영어 단어가 존재한다는 사실에 놀랐다. 대표적인 예로 AB (Abdominal의 약자), BAA (양의 울음 소리), AA (용암의 종류), ABBA (스웨덴 팝 그룹)이 있다.

이런 사실에 놀란 수빈이는 간단한 게임을 만들기로 했다. 두 문자열 S와 T가 주어졌을 때, S를 T로 바꾸는 게임이다. 문자열을 바꿀 때는 다음과 같은 두 가지 연산만 가능하다.

문자열의 뒤에 A를 추가한다.
문자열을 뒤집고 뒤에 B를 추가한다.
주어진 조건을 이용해서 S를 T로 만들 수 있는지 없는지 알아내는 프로그램을 작성하시오. 

##### 입력
첫째 줄에 S가 둘째 줄에 T가 주어진다. (1 ≤ S의 길이 ≤ 999, 2 ≤ T의 길이 ≤ 1000, S의 길이 < T의 길이)

##### 출력
S를 T로 바꿀 수 있으면 1을 없으면 0을 출력한다.



#### **풀이**
** [잘못된 접근]**

처음에는 S값에서 T를 만들어 가는 방식으로 접근했다.
그래서 총 4가지를 고려해야 한다고 생각했다.
    1. 현재 문자열 뒤에 "A"를 추가하였을 때, 같은 문자열 수만큼 T와 같은가
        ->S 문자열 뒤에 "A" 추가
    2. 현재 문자열을 뒤집고 "B"를 추가하였을 때, 같은 문자열 수만큼 T와 같은가
        ->S 문자열 뒤에 "B" 추가
    3. 현재 문자열 뒤에 "A"를 추가하고, 문자열을 뒤집은 후에 B를 추가하면 같은 문자열 수만큼 T와 같은가
        ->S 문자열 뒤에 "A" 추가(뒤에 부분은 2번에서 처리)
    4. 현재 문자열을 뒤집고 "B"를 추가한 후, 문자열 뒤에 "A"를 추가하면 같은 문자열 수만큼 T와 같은가
        ->S 문자열 뒤에 "B" 추가(뒤에 부분은 1번에서 처리)

이렇게 정리를 하고 코드를 구현하였는데, 계속 예외가 있어서 처음부터 다시 생각했다.

** [해결] **

내가 너무 복잡하게 생각했다.
그냥 문자열 T에서 거꾸로 타고 올라가서 문자열 S인지 확인하면 되는 것이었다.
    1. 문자열 T[-1]이 B이면 T[-1]을 제거한 후, 문자열 뒤집음
    2. 문자열 T[-1]이 A이면 T[-1]을 제거

** [코드] **

    S = input()
    T = input()
    for num in range(len(T)-len(S)):
        if T[-1] == "A":
            T = T[:-1]
        else:
            T = T[:-1]
            T = T[::-1]
    if S == T:
