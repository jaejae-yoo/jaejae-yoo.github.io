> ## Node.js

### Node란 무엇인가?
- JavaScipt를 이용해서 서버를 개발할 수 있는 도구

### 왜 Node인가?
- Node 사용 이전에는 웹 서버에 파일을 업로드할 때, 업로드가 완료되기 전까지 웹 서버에서 다른 작업을 할 수 없었음 
    
    ➡ 이러한 불편함을 해소하기 위해 만든 것이 Node

### Node의 비동기식 입출력 방식
    //B.txt에는 알파벳 B가 존재
    var fs = require('fs');
**동기식 방법**

    console.log('A');
    var result = fs.readFileSync('B.txt', 'utf8');
    console.log(result);
    console.log('C');
    // 실행 결과 : A B C

**비동기식 방법**

    console.log('A');
    fs.readFile('B.txt', 'utf8', function(err, result){
        console.log(result);
    });
    console.log('C');
    // 실행 결과 : A C B

**비동기 방식**은 하나의 요청 처리가 끝날 때까지 기다리지 않고 동시에 처리할 수 있는 방식입니다.

### Callback 함수
프로그램에서 해당 파일의 내용을 처리할 수 있는 시점이 되면 **콜백 함수**가 호출됩니다. 따라서 프로그램이 파일 읽기 작업이 끝날 때까지 대기하지 않아도, 파일을 다 읽은 시점에 통보를 받고 파일의 내용을 화면에 보여주는 작업을 진행할 수 있습니다.
